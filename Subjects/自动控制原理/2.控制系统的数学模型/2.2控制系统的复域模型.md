# **复域系统--传递函数**
### 传递函数的定义
-  ![[线性定常系统微分方程一般形式|200]]
- 在 $0$ 初始条件下, 线性定常系统输出量拉氏变换与输入量拉氏变换之比
$$G(s) = \frac{C(s)}{R(s)}$$
### 传递函数的标准形式
- 微分方程一般形式
	$a_n c^{(n)} + a_{n-1} c^{(n-1)} + ... +a_1 c' + a_0 c = b_m r^{(m)} + b_{m-1} r^{(m-1)} + ... + b_1 r' + b_0 r$
- 拉氏变换
	$(a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0)C(s) = (b_m s^m + b_{m-1} s^{m-1} + ... + b_1 s + b_0)R(s)$
- 传递函数
	$G(s) = \frac{C(s)}{R(s)} = \frac{b_m s^m + b_{m-1} s^{m-1} + ... + b_1 s + b_0}{a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0}$
	- 首 $1$ 标准型
		$G(s) = \frac{b_m}{a_n} \cdot \frac{s^m + \frac{b_{m-1}}{b_m}s^{m-1}+...+\frac{b_1}{b_m}s + \frac{b_0}{b_m}}{s^n + \frac{a_{n-1}}{a_n}s^{n-1}+...+ \frac{a_1}{a_n}s + \frac{a_0}{a_n}} \overset{K^* = \frac{b_m}{a_n}}{=} K^* \frac{\displaystyle \prod_{j = 1}^{m}(s - z_j)}{\displaystyle \prod_{i = 1}^{n}(s - p_i)}$
	- 尾 $1$ 标准型 (分子分母首项为正)
		$G(s) = \frac{b_0}{a_0} \cdot \frac{\frac{b_m}{b_0}s^m + \frac{b_{m-1}}{b_0}s^{m-1}+...+\frac{b_1}{b_0}s + 1} {\frac{a_n}{a_0}s^n + \frac{a_{n-1}}{a_0}s^{n-1}+...+ \frac{a_1}{a_0}s + 1} \overset{增益K = \frac{b_0}{a_0}}{=} K \frac{\displaystyle \prod_{k = 1}^{m_1}(\tau_k s + 1)\prod_{l = 1}^{m_2}(\tau_l^2 s^2 + 2\xi \tau_l s + 1)}{{\LARGE s^v} \displaystyle \prod_{i = 1}^{n_1}(T_i s + 1) \prod_{j = 1}^{n_2}(T_j^2 s^2 + 2\xi T_j s + 1)}$
### 传递函数的性质
1. $G(s)$ 是复函数
2. $G(s)$ 只与系统自身的结构参数有关
3. $G(s)$ 与系统微分方程等价
4. $G(s) = L[k(t)]$ 传递函数与系统的单位脉冲响应互为拉氏变换对
5. $G(s)$ 与 $s$ 平面上的零点极点图相对应
### 传递函数的局限性
1. 原则上不反映非零初始条件时系统响应的全部信息
2. 适合于描述单输入、单输出系统
3. 只能用于表示定常系统

## 典型环节
| 名称       | 微分方程                                                 | 传递函数                              | 例子                      |
| -------- | ---------------------------------------------------- | --------------------------------- | ----------------------- |
| 比例环节     | $c = K \cdot r$                                      | $K$                               | 电位器, 放大器, 减速器, 测速发电机等   |
| 惯性环节     | $T\dot{c}+c = r$                                     | $\frac{1}{Ts+1}$                  | $R-C$ 电路, 交、直流电动机       |
| 震荡环节     | $T^2 \ddot{c}+2\zeta T\dot{c}+c = r$</br>$0<\zeta<1$ | $\frac{1}{T^2 s^2 + 2\zeta Ts+1}$ | $R-L-C$ 电路, 弹簧-质块-阻尼器系统 |
| 积分环节     | $\dot{c} = r$                                        | $\frac{1}{s}$                     | 电容上的电流与电压, 测速发电机位移与电压   |
| 微分环节     | $c = \dot{r}$                                        | $s$                               |                         |
| 一阶复合微分环节 | $c = \tau \dot{r} + r$                               | $\tau s + 1$                      |                         |
| 二阶复合微分环节 | $c = \tau^2\ddot{r}+2\tau \zeta \dot{r}+r$           | $\tau^2 s^2 +2\zeta \tau s+1$     |                         |
- 不同元部件可以有相同形式的传递函数
- 当输入输出变量选择不同时, 同一元部件可以有不同的传递函数
- 任一传递函数都可以看作典型环节的组合
