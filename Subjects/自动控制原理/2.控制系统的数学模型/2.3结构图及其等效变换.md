## **结构图的组成及绘制**
系统结构图是描述组成系统的各元部件之间信号传递关系的图形化数学模型

### 例1 $X-Y$ 记录仪
| 环节                              | 微分方程组                             | 结构图                                                                                                                                  |
| ------------------------------- | --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| 反馈口                             | $\Delta v = u_r - u_p - u_\omega$ | $\Delta V(s) = U_r(s)-U_p(s)-U_\omega(s)$                                                                                            |
| 放大器                             | $u = K_1 \cdot \Delta v$          | $U_(s) = K_1 \cdot \Delta V(s)$                                                                                                      |
| 电动机                             | $T_m \omega + \omega = K_m u$     | $T_m \cdot s\Omega(s)+\Omega(s) = K_m \cdot U(s)$                                                                                    |
| 减速器                             | $\theta_2 = K_2 \theta_m$         | $\Theta_2(s) = K_2 \cdot \Theta_m(s)$                                                                                                |
| 绳轮                              | $L = K_3\theta_2$                 | $L(s) = K_3 \cdot \Theta_2(s)$                                                                                                       |
| 测速机                             | $u_\omega = K_\omega \omega$      | $U_\omega(s) = K_\omega \cdot \Omega(s)$                                                                                             |
| 电桥                              | $u_p = K_4 L$                     | $U_p(s) = K_4\cdot L(s)$                                                                                                             |

### 例2 电枢控制式直流电机
| 电枢回路                            | $u_a = Ri + E_b$                                     | $U_a(s)=R\cdot I(s)+E_b(s)$                                                                                                          |
| ------------------------------- | ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| 电枢反电势                           | $E_b = c_e \cdot \omega_m$                           | $E_b(s) = c_e \cdot \Omega_m (s)$                                                                                                    |
| 电磁力矩                            | $M_m = c_m i$                                        | $M_m(s) = c_m \cdot I(s)$                                                                                                            |
| 力矩平衡                            | $J_m \overset{\cdot}{\omega}_m + f_m \omega_m = M_m$ | $J_m\cdot s\Omega_m(s)+f_m\cdot \Omega_m(s) = M_m(s)$                                                                                |
| ![[Image_1792915827455833.jpg]] | <                                                    | 等效化简 $\overset{U_a}{\longrightarrow} \frac{\frac{c_m}{Rf_m+c_mc_e}}{\frac{RJ_m}{Rf_m+c_mc_e}s+1}\overset{\Theta_m}{\longrightarrow}$ |
