## **系统的开环传递函数**
人为断开系统的反馈通路，将前向通路与反馈通路上的传递函数乘在一起，称为开环传递函数，表示为 $G(s)H(s)$
此处的开环传递函数是针对闭环系统而言的，而非开环系统的传递函数。
化为尾 $1$ 标准型，系数称为开环增益
$图中:G(s)H(s) = G_1(s)G_2(s)H(s)$
## **闭环系统的传递函数**

|         | 闭环传递函数                                                                                  | 误差传递函数                                                                                        |
| ------- | --------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| 控制输入作用下 | $\Phi(s) = \frac{C(s)}{R(s)} \overset{图中}{=} \frac{G_1(s)G_2(s)}{1+G_1(s)G_2(s)H_1(s)}$ | $\Phi_e(s) = \frac{E(s)}{R(s)} \overset{图中}{=} \frac{1}{1+G_1(s)G_2(s)H_1(s)}$                |
| 干扰作用下   | $\Phi_n(s) = \frac{C(s)}{N(s)} \overset{图中}{=} \frac{G_2(s)}{1+G_1(s)G_2(s)H_1(s)}$     | $\Phi_{en}(s) = \frac{E(s)}{N(s)} \overset{图中}{=} \frac{-G_2(s)H_1(s)}{1+G_1(s)G_2(s)H_1(s)}$ |
| 总输出     | $C(s) = \Phi(s)R(s)+\Phi_n(s)N(s)$                                                      | $E(s) = \Phi_e(s)R(s)+\Phi_{en}(s)N(s)$                                                       |
