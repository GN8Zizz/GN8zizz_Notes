# **时域系统--微分方程**
### 线性定常系统微分方程的一般形式
-  ![[线性定常系统微分方程一般形式|200]]
-  $a_n \frac {{\rm d}^n c(t)}{{\rm d}t^n} + a_{n-1} \frac{{\rm d}^{n-1} c(t)}{{\rm d}t^{n-1}} + ... +a_1 \frac{{\rm d} c(t)}{{\rm d}t} + a_0 c(t) = b_m \frac {{\rm d}^m r(t)}{{\rm d}t^m} + b_{m-1} \frac{{\rm d}^{m-1} r(t)}{{\rm d}t^{m-1}} + ... +b_1 \frac{{\rm d} r(t)}{{\rm d}t} + b_0 r(t)$
- 注:
	- 看变量判断线性: 每项仅与输入输出自身或自身导数有关, 不含常数项、高次项、交叉相乘项, 满足该形式的微分方程一定满足叠加原理, 即线性系统
	- 看系数判断定常: $a_n、b_m$ 全部都是常数, 非时间的函数, 定常系统
#### 例1: $R-L-C$ 串连电路
![[RLC电路模型|400]]
$\left\{\begin{array}{l} i_C = C\frac{{\rm d}u_C}{{\rm d} t} \\ u_L = L\frac{{\rm d}i_L}{{\rm d}t} \end{array}\right.$
#### 例2:弹簧--阻尼器系统
![[弹簧阻尼器系统|300]]
$\left\{\begin{array}{l} 弹簧:F = k \Delta x \\ 阻尼器:F = f \Delta v \end{array}\right.$
#### 例3:电枢控制式直流电动机
![[电动机|300]]
$\left\{\begin{array}{l} 电枢回路:u_a = Ri + E_b \\ 电枢反电动势:E_b = C_e \omega_m \\ 电磁力矩:M_m = C_m i \\ 力矩平衡:J_m \overset{\cdot}{\omega_m} + f_m \omega_m = M_m \end{array}\right.$
$\Rightarrow T_m \overset{\cdot}{\omega_m} + \omega_m = K_a u_a \hspace{2mm}或\hspace{2mm}T_m \overset{\cdot \cdot}{\theta_m} + \overset{\cdot}{\theta_m} = K_a u_a$
$电动机的机电时间常数: T_m = \frac{R J_m}{R f_m + C_m C_e};电动机的传动系数: K_a = \frac{C_m}{R f_m + C_m C_e}$

## **线性定常微分方程求解**
![[拉氏变换|250]]
### 拉氏变换相关知识
1. 复数有关概念
	1. 复数: $s = \sigma + j \omega$
		复函数: $F(s) = F_x(s) + jF_y(s)$
	2. 模: $|F(s)| = \sqrt{F_x^2 + F_y^2}$
		相角: $\angle F(s) = {\rm arctan}\frac{F_y}{F_x}$
	3. 共轭: $\overline{F(s)} = F_x - j F_y$
	4. 解析: 若 $F(s)$ 在 $s$ 点的各阶导数都存在, 则 $F(s)$ 在点 $s$ 解析
2. 拉氏变换的定义
	-  $L[f(t)] = F(s) = \int_0^\infty f(t) \cdot e^{-st} {\rm d}t$ 
	-  $F(s)$ 像
		$f(t)$ 原像
3. 常见函数拉氏变换
	
|       | $f(t)$              | $F(s)$                          |     |
| ----- | ------------------- | ------------------------------- | --- |
| 单位脉冲  | $\delta (t)$        | $1$                             |     |
| 单位阶跃  | $1(t)$              | $\frac{1}{s}$                   |     |
| 单位斜坡  | $t$                 | $\frac{1}{s^2}$                 |     |
| 单位加速度 | $\frac{t^2}{2}$     | $\frac{1}{s^3}$                 |     |
| 指数函数  | $e^{-at}$           | $\frac{1}{s+a}$                 |     |
| 正弦函数  | ${\rm sin}\omega t$ | $\frac{\omega}{s^2 + \omega^2}$ |     |
| 余弦函数  | ${\rm cos}\omega t$ | $\frac{s}{s^2 + \omega^2}$      |     |
| 方波函数  |                     |                                 |     |
4. 拉氏变换重要定理

| 线性性质        | $L[af_1(t) \pm bf_2(t)] = aF_1(s) \pm bF_2(s)$                                                                                                                                           |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 微分定理        | $L[f(t)] = F(s)$<br>$L[f'(t)] = s F(s) - f(0)$<br>$L[f''(t)] = s^2 F(s) - sf(0) - f'(0)$<br>$L[f''(t)] = s^3 F(s) - s^2f(0) - sf'(0) - f''(0)$<br>$0$ 初始条件下有:<br>$L[f^{(n)}] = s^n F(s)$ |
| 积分定理        | $L[\int f(t {\rm d}t)] = \frac{1}{s}F(s) + \frac{1}{s} f^{(-1)}(0)$ <br>$0$ 初始条件下有:<br>$L[\int f(t {\rm d}t)] = \frac{1}{s}F(s)$                                                         |
| 实位移定理       | $L[f(t - \tau_0)] = e^{-\tau_0 \cdot s} \cdot F(s)$                                                                                                                                      |
| 复位移定理       | $L[e^{A\cdot t} f(t)] = F(s - A)$                                                                                                                                                        |
| 初值定理        | $\displaystyle \lim_{t \rightarrow 0}f(t) = \lim_{s \rightarrow \infty} s \cdot F(s)$<br>用于计算系统响应初始值                                                                                     |
| 终值定理(终值存在时) | $\displaystyle \lim_{t \rightarrow \infty} f(t) = \lim_{s \rightarrow 0} s \cdot F(s)$<br>用于计算系统稳态误差                                                                                     |
5. 拉氏反变换
	1. 反演公式
		$f(t) = \frac{1}{2\pi j} \displaystyle \int_{\sigma - j\infty}^{\sigma + j\infty} F(s) \cdot e^{ts} {\rm d}s$
	2. 查表法(分解部分分式法) $\left\{\begin{array}{l} 试凑法 \\ 系数比较法 \\ 留数法 \end{array}\right.$
### 用 $L$ 变换法解线性常微分方程
-  $\left.\begin{array}{l} &\hspace{7mm} a_n c^{(n)} + a_{n-1} c^{(n-1)} + ... +a_1 c' + a_0 c = b_m r^{(m)} + b_{m-1} r^{(m-1)} + ... + b_1 r' + b_0 r \hspace{2mm} (0初条件、n>m)\\ &\overset{L}{\Rightarrow} (a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0)C(s) = (b_m s^m + b_{m-1} s^{m-1} + ... + b_1 s + b_0)R(s) \\ &\Rightarrow C(s) = \frac{b_m s^m + b_{m-1} s^{m-1} + ... + b_1 s + b_0}{a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0}R(s) \\ &\overset{r(t) = \delta(t)}{\Rightarrow} \frac{b_m s^m + b_{m-1} s^{m-1} + ... + b_1 s + b_0}{a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0} = \frac{C_1}{s - \lambda_1} + \frac{C_2}{s - \lambda_2} + ... + \frac{C_n}{s - \lambda_n} (其中\lambda_i 为特征根(极点)) \\ &\overset{L^{-1}}{\Rightarrow} c(t) = L^{-1}[C(s)] = C_1 e^{\lambda_1 t} + C_2 e^{\lambda_2 t} +v ... + C_n e^{\lambda_n t} (其中e^{\lambda_i t}为相对于\lambda _i的模态)\end{array}\right.$
- 系统对单位脉冲激励的响应是对应特征根模态的线性组合
- 用留数法分解部分分式
	$\left.\begin{array}{l} 一般有F(s) = \frac{B(s)}{A(s)} = \frac{b_m s^m + b_{m-1} s^{m-1} + ... + b_1 s + b_0}{a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0} \hspace{2mm} (n>m) \\ 设A(s) = a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0 = (s - p_1)(s - p_2)...(s - p_n) \\ 1. \hspace{2mm}当A(s) = 0无重根时 \\ \qquad F(s) = \frac{C_1}{s - p_1} + \frac{C_2}{s - p_2} + ... + \frac{C_n}{s - p_n} = {\displaystyle \sum_{i = 1}^{n} \frac{C_i}{s - p_i}} \\ \qquad 其中: \left\{\begin{array}{l} C_i = \displaystyle \lim_{s \rightarrow p_i}F(s) = \lim_{s \rightarrow p_i}(s - p_i)\frac{b_m s^m + b_{m-1} s^{m-1} + ... + b_1 s + b_0}{(s - p_1) ... (s - p_i) ... (s - p_n)} \\ C_i = \frac{B(s)}{A'(s)}\Bigg|_{s = p_i} = \frac{b_m s^m + b_{m-1} s^{m-1} + ... + b_1 s + b_0}{\displaystyle \lim_{s \rightarrow p_i} \frac{(s - p_1)...(s - p_i)...(s - p_n)}{(s - p_i)}} \end{array}\right. \\ \qquad f(t) = C_1 e^{p_1 t} + C_2 e^{p_2 t} + ... + C_n e^{p_n t} = \displaystyle \sum_{i = 1}^{n} C_i e^{p_i t} \\ 2.\hspace{2mm}当A(s) = (s - p_1)...(s - p_n)有重根时,设p_1为m重根,其余为单根 \\ \qquad F(s) = \frac{C_m}{(s - p_1)^{m}} + \frac{C_{m-1}}{(s - p_1)^{m-1}} + ... + \frac{C_1}{s - p_1} + \frac{C_{m+1}}{s-p_{m+1}} + ... + \frac{C_n}{s-p_n} \\ \qquad 其中:\left\{\begin{array}{l}  C_m = \displaystyle \lim_{s \rightarrow p_1}(s-p_1^m) F(s) \\ C_{m-1} = \frac{1}{1!} \displaystyle \lim_{x \rightarrow p_1}\frac{{\rm d}}{{\rm d}s} [(s - p_1)^m F(s)] \\...\\ C_{m-j} = \frac{1}{j!}\displaystyle \lim_{s \rightarrow p_1}\frac{{\rm d}^{(j)}}{{\rm d}s^{j}}[(s-p_1)^m F(s)] \\ ... \\C_1 = \frac{1}{(m-1)!} \displaystyle \lim_{s \rightarrow p_1}\frac{{\rm d}^{(m-1)}}{{\rm d}s^{m-1}}[(s-p_1)^m F(s)] \end{array}\right. \\ \qquad f(t) = L^{-1}[\frac{C_{m}}{(s-p_1)^{m}} + \frac{C_{m-1}}{(s-p_1)^{m-1}} + ... + \frac{C_{2}}{(s-p_1)^{2}} + \frac{C_{1}}{(s-p_1)}] + [\frac{C_{m+1}}{s - p_{m+1}} + ... + \frac{C_n}{s - p_n}]  \\ \qquad \hspace{9.5mm} = [\frac{C_{m}}{(m-1)!}t^{m-1} + \frac{C_{m-1}}{(m-2)!}t^{m-2} + ... + C_2t + C_1]e^{p_1 t} + \displaystyle \sum_{i = m+1}^n C_ie^{p_it} \end{array}\right.$
