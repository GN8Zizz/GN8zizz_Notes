- 稳态误差是系统的性能指标，是对系统控制精度的度量。
	对稳定的系统研究稳态误差才有意义，所以计算稳态误差以系统稳定为前提。
	本讲只套路原理性误差。
- 通常把在<mark class="hltr-grey">阶跃输入</mark>作用下没有原理性稳态误差的系统称为“无差系统”，有原理性误差的系统称为“有差系统”。

### 稳态误差
$稳态误差\left\{\begin{array}{l} 静态误差(终值误差)：e_{ss} = \displaystyle\lim_{t \rightarrow}e(t) = e(\infty)\\ 动态误差：误差中的稳态分量e_s(t)，时间趋于无穷大时，反应稳态误差随时间的变化规律 \end{array}\right.$ 
## **计算稳态误差的一般方法**
1. 判定系统稳定性
2. 求误差传递函数
	$\Phi_e(s) = \frac{E(s)}{R(s)}、\Phi_{en}(s) = \frac{E(s)}{N(s)}$
3. 用终值定理求稳态误差
	$e_{ss} = \displaystyle\lim_{s\rightarrow 0}s[\Phi_e(s)R(s)+\Phi_{en}(s)N(s)]$
	$e_{ss}\left\{\begin{array}{l} 与系统自身的结构参数有关 \\ 与外作用的类型有关 \end{array}\right.$

$K$：开环增益
$v$：型别(开环传递函数中纯积分环节的个数)
- $e_{ss} = \displaystyle \lim_{s\rightarrow0}s\Phi_e(s)R(s) = \displaystyle\lim_{s\rightarrow0}sR(s)\frac{1}{1+G_1(s)H(s)} = \displaystyle\lim_{s\rightarrow0}sR(s)\frac{1}{1+\frac{K}{s^v}G_0(s)}$
	- $对r(t) = A\cdot 1(t)$
		$\left.\begin{array}{l}e_{ssp} = \displaystyle\lim_{s\rightarrow0}s\Phi_e(s)R(s) = \displaystyle\lim_{s\rightarrow0}s\frac{A}{s}\frac{1}{1+G_1(s)H(s)} = \frac{A}{1+\displaystyle\lim_{s\rightarrow0}G_1(s)H(s)} = \frac{A}{1+K_p} \\ 静态位置误差参数：K_p = \displaystyle\lim_{s\rightarrow0}G_1(s)H(s) = \displaystyle\lim_{s\rightarrow}\frac{K}{s^v} \end{array}\right.$
	- $对r(t) = A\cdot t$
		$\left.\begin{array}{l} e_{ssv} = \displaystyle\lim_{s\rightarrow0}s\Phi_e(s)R(s) = \displaystyle\lim_{s\rightarrow0}s\frac{A}{s^2}\frac{1}{1+G_1(s)H(s)} = \frac{A}{\displaystyle\lim_{s\rightarrow0}sG_1(s)H(s)} = \frac{A}{K_v} \\ 静态速度误差参数：K_v = \displaystyle\lim_{s\rightarrow0}sG_1(s)H(s) = \displaystyle\lim_{s\rightarrow}\frac{K}{s^{v+1}} \end{array}\right.$
	- $对r(t) = \frac{A}{2}t^2$
		$\left.\begin{array}{l} e_{ssa} = \displaystyle\lim_{s\rightarrow0}s\Phi_e(s)R(s) = \displaystyle\lim_{s\rightarrow0}s\frac{A}{s^3}\frac{1}{1+G_1(s)H(s)} = \frac{A}{\displaystyle\lim_{s\rightarrow0}s^2G_1(s)H(s)} = \frac{A}{K_a} \\ 静态加速度误差参数：K_a = \displaystyle\lim_{s\rightarrow0}s^2G_1(s)H(s) = \displaystyle\lim_{s\rightarrow}\frac{K}{s^{v+2}} \end{array}\right.$

## **动态误差系数法**
用静态误差系数法只能求出误差的稳态值 $e_{ss} = \displaystyle\lim_{t\rightarrow\infty}e(t)$，而稳态误差随着时间变化的规律无法获得；用动态误差系数法可以研究误差中的稳态分量 $e_{s}(t)$ 随时间的变化规律。
$\left.\begin{array}{l} \Phi_e(s) &= \frac{E(s)}{R(s)} = \Phi_e(0)+\frac{1}{1!}\Phi_e'(0)s+\frac{1}{2!}\Phi_e''(0)s^2+...+\frac{1}{i!}\Phi_e^{(i)}(0)s^i+... \\ &\downarrow令C_i = \frac{1}{i!}\Phi_e^{(i)}(0)\quad i = 0,1,2,... \quad称为动态误差系数 \\ &=C_0 + C_1s + C_2s^2 + ... = \displaystyle\sum_{i = 0}^{\infty}C_is^i \\ &\downarrow \\ E(s) &= \Phi_e(s)R(s) \\&=C_0R(s)+C_1R(s)+C_2s^2R(s)+....+C_is^iR(s)+... \quad对常用的t的多项式来说，此表达式非无限多\\ &\downarrow L_{-1}\\ e_s(t) &= C_0r(t)+C_1\overset{\cdot}{r(t)}+C_2\overset{\cdot\cdot}{r(t)}+...+C_ir^{(i)}(t)+... = \displaystyle\sum_{i=0}^{\infty}C_ir^{(i)}(t) \end{array}\right.$
- 求 $C_i$ 的方法
	1. 系数比较法
	2. 长除法
