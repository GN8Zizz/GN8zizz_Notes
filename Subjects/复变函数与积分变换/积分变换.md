# 引入
- 积分变换：
	- 通过 $\displaystyle\int_{a}^{b}f(t)\cdot k(t,\alpha){\rm d}t$，使 $f(t)\rightarrow F(\alpha)$
	- 其中 $f(t)$：象原函数、$F(\alpha)$：象函数、$k(t,\alpha)$：核函数
- Fourier变换：
	- $(a,b)\rightarrow(-\infty,+\infty)\quad k(t,\omega) = e^{-j\omega t}$
		$\displaystyle F(\omega) = \int_{-\infty}^{+\infty}f(t)\cdot e^{-j\omega t}{\rm d}t$
		时域 $\rightarrow$ 频域
- Laplace变换：
	- $(a,b)\rightarrow (0,+\infty)\quad k(t,s) = e^{-st}$
		$\displaystyle F(s) = \int_{0}^{+\infty}f(t)\cdot e^{-st}{\rm d}t$ 
		时域 $\rightarrow$ 复域

# Fourier级数
$Fourier级数\rightarrow Fourier积分$
$\hspace{3mm}周期函数\hspace{3.5mm} \rightarrow \hspace{3.5mm}非周期$
$\hspace{11mm}\sum\hspace{11.5mm}\rightarrow\hspace{10mm}\int$
- Fourier级数：
	条件：
	$\displaystyle f(t) = f(t+T)\quad\omega = \frac{2\pi}{T}$
	$Dirichlet条件:\left\{\begin{array}{l} 1.有限个第一类间断点(有限次跳跃) \\ 2.有限个极值点(有限次拐弯) \end{array}\right.$
	- $\displaystyle f(t) = \frac{a_0}{2}+\sum_{n=1}^{\infty}(a_n{\rm cos}n\omega t+b_n{\rm sin}n\omega t)$
	- "="成立条件：
		1. $f(t)$ 连续
		2. $f(t)$ 在 $t$ 间断，右级数 $\displaystyle =\frac{f(t^-)+f(t^+)}{2}$
- Fourier级数的指数形式：
	由Euler公式：$e^{iz} = {\rm cos}z+i{\rm sin}z$
	指数形式：$f(t) = \displaystyle\sum_{-\infty}^{+\infty}C_ne^{in\omega t}$
	- 系数 $C_n$ 求解：
		- 同乘 $e^{-jm\omega t}$，在 $[\displaystyle-\frac{T}{2},\frac{T}{2}]$ 积分
			$\left.\begin{array}{l} \displaystyle\int_{-\frac{T}{2}}^{\frac{T}{2}}f(t)e^{-jm\omega t}{\rm d}t &= \displaystyle\sum_{n=-\infty}^{+\infty}C_n\int_{-\frac{T}{2}}^{\frac{T}{2}}e^{jn\omega t}e^{-jm\omega t}{\rm d}t \\ &= \displaystyle\sum_{n\ne m}C_n\int_{-\frac{T}{2}}^{\frac{T}{2}}e^{j(n-m)\omega t}{\rm d}t + C_mT\qquad 而\displaystyle\sum_{n\ne m}C_n\int_{-\frac{T}{2}}^{\frac{T}{2}}e^{j(n-m)\omega t}{\rm d}t=0  \\ &= C_mT \end{array}\right.$

# Fourier积分的定义
- 条件：
	- $Dirichlet条件:\left\{\begin{array}{l} 1.有限个第一类间断点(有限次跳跃) \\ 2.有限个极值点(有限次拐弯) \end{array}\right.$
	- $\displaystyle\int_{-\infty}^{+\infty}|f(t)|{\rm d}t$ 有限
- 定义：
	-  $\displaystyle F(\omega) = \int_{-\infty}^{+\infty}f(\tau)e^{-j\omega \tau}{\rm d}\tau$
	-  $\displaystyle f(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty}F(\omega)e^{j\omega t}{\rm d}\omega$
	- 特别的：
		- $f(t)$ 为偶函数
			$f(t)$ 的余弦积分表示：
			$f(t) = \displaystyle\frac{2}{\pi}\int_{0}^{+\infty}[\int_{0}^{+\infty}f(\tau){\rm cos}\omega\tau{\rm d}\tau]{\rm cos}\omega t{\rm d}t$
			$\displaystyle F_{{\rm cos}}(\omega)=\int_{0}^{+\infty}f(\tau){\rm cos}\omega\tau{\rm d}\tau$
		- $f(t)$ 为奇函数
			$f(t)$ 的正弦积分表示：
			$f(t) = \displaystyle\frac{2}{\pi}\int_{0}^{+\infty}[\int_{0}^{+\infty}f(\tau){\rm sin}\omega\tau{\rm d}\tau]{\rm sin}\omega t{\rm d}t$
			$\displaystyle F_{\rm sin}(\omega) = \int_{0}^{+\infty}f(\tau){\rm sin}\omega\tau{\rm d}\tau$

# 矩形脉冲函数的Fourier积分
对矩形脉冲函数：$f(t) = \left\{\begin{array}{l} 1\quad|t|\le1 \\ 0\quad其他 \end{array}\right.$
$\displaystyle F(\omega) = \int_{-\infty}^{+\infty}f(t)e^{-j\omega t}{\rm d}t = 2\frac{{\rm sin}\omega}{\omega}$
$\displaystyle f(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty}F(\omega)e^{j\omega t}{\rm d}\omega = \frac{2}{\pi}\int_{0}^{+\infty}\frac{{\rm sin}\omega{\rm cos}\omega t}{\omega}{\rm d} \omega = \left\{\begin{array}{l} 1\quad -1<t<1 \\ \frac{1}{2}\quad t = \pm 1 \\ 0\quad\hspace{1.5mm} t>1或t<-1 \end{array}\right.$
$t = 0$ 时有：
Dirichlet积分：$\displaystyle\int_{0}^{+\infty}\frac{{\rm sin}\omega}{\omega}{\rm d}\omega = \frac{\pi}{2}$

# Fourier变换的性质
$f(t)\mathop{\longleftrightarrow}\limits^{\mathcal F}F(\omega)$
1. 线性性质：
	${\mathcal F}[\alpha f(t)+\beta g(t)] = \alpha{\mathcal F}[f(t)]+\beta{\mathcal F}[g(t)]$
2. 位移性质：
	$f(t\pm t_0)\mathop{\longleftrightarrow}\limits^{\mathcal F} e^{\pm j\omega t_0}F(\omega)$
	$e^{\mp j\omega_0 t}f(t)\mathop{\longleftrightarrow}\limits^{\mathcal F}F(\omega\pm\omega_0)$
3. 微分性质：前提：$\displaystyle\lim_{|t|\rightarrow\infty}f(t) = 0$
	$f^{(n)}(t) \mathop{\longleftrightarrow}\limits^{\mathcal F}\hspace{1mm} (j\omega)^nF(\omega)$
	$(-jt)^n f(t)\mathop{\longleftrightarrow}\limits^{\mathcal F}F^{(n)}(\omega)\Rightarrow t^nf(t)\mathop{\longleftrightarrow}\limits^{\mathcal F}j^n F^{(n)}(\omega)$
4. 积分性质：前提：$t\rightarrow\infty, \displaystyle\int_{-\infty}^{t}f(t){\rm d}t\rightarrow0$
	$\displaystyle\int_{-\infty}^{t}f(t){\rm d}t \mathop{\longleftrightarrow}\limits^{\mathcal F}\frac{1}{j\omega}F(\omega)$
5. 相似性质：
	$\displaystyle f(at)\mathop{\longleftrightarrow}\limits^{\mathcal F}\frac{1}{|a|}F(\frac{\omega}{a})$
6. 能量积分：
	内积：$\left\{\begin{array}{l} \displaystyle 时域:[f_1(t),f_2(t)] \overset{\tiny\triangle}{=}\int_{-\infty}^{+\infty}f_1(t)\overline{f_2(t)}{\rm d}t \\ \displaystyle 频域:[F_1(\omega),F_2(\omega)] \overset{\tiny\triangle}{=}\frac{1}{2\pi}\int_{-\infty}^{+\infty}F_1(\omega)\overline{F_2(\omega)}{\rm d}\omega \end{array}\right.$
	长度：$\left\{\begin{array}{l} \|f_1\|^2 = [f_1,f_2] \hspace{4mm}\qquad时域信号能量 \\ \|F_1(\omega)\|^2 = [F_1,F_2]\quad频域信号能量 \end{array}\right.$
	$\|f\|^2 = \|F(\omega)\|^2$

# 卷积
$f_1(t)\ast f_2(t) = \displaystyle\int_{-\infty}^{+\infty}f_1(\tau)f_2(t-\tau){\rm d}\tau$
- 性质
	- 交换律
		$f_1\ast f_2 = f_2\ast f_1$
	- 结合律
		$(f_1\ast f_2)\ast f_3 = f_1\ast (f_2\ast f_3)$
	- 分配率
		$(f_1+f_2)\ast f_3 = f_1\ast f_3 + f_2\ast f_3$
	- 微分性质
		$\displaystyle \frac{{\rm d}(f_1\ast f_2)}{{\rm d}t} = f_1\ast \frac{{\rm d}f_2}{{\rm d}t} = \frac{{\rm d}f_1}{{\rm d}t}\ast f_2$
	- 积分性质
		$\displaystyle\int_{-\infty}^{t}(f_1\ast f_2){\rm d}\xi = \int_{-\infty}^{t}f_1{\rm d}\xi \ast f_2 = f_1 \ast\int_{-\infty}^{t}f_2 {\rm d}\xi$
	- 单位 $1$ ：$\delta(t)$
		$f(t)\ast \delta(t) = f(t)$
	- 导数写为卷积
		$f'(t) = f(t)\ast \delta'(t)$
	- 积分写为卷积
		$\displaystyle\int_{-\infty}^{t}f(\tau){\rm d}\tau = f(t)\ast u(t)$

# 卷积定理
## Fourier
${\mathcal F}[f_1\ast f_2] = {\mathcal F}[f_1]\cdot{\mathcal F}[f_2]$
${\mathcal F}^{-1}[F_1(\omega)\ast F_2(\omega)] = 2\pi f_1f_2$
即
$f_1\ast f_2\mathop{\longleftrightarrow}\limits^{\mathcal F}F_1(\omega)F_2(\omega)$
$2\pi f_1f_2\mathop{\longleftrightarrow}\limits^{\mathcal F}F_1(\omega)\ast F_2(\omega)$
## Laplace
${\mathcal L}[f_1\ast f_2] = {\mathcal L}[f(1)]\cdot {\mathcal L}[f(2)]$
${\mathcal L}^{-1}[F_1(s)F_2(s)] = f_1 \ast f_2$

# Laplace变换的性质
$f(t) \mathop{\longleftrightarrow}\limits^{\mathcal L}F(s)$
1. 线性性质：
	${\mathcal L}[\alpha f(t)+\beta g(t)] = \alpha{\mathcal L}[f(t)]+\beta{\mathcal L}[g(t)]$
2. 位移(延时)性质：前提：$t<t_0$时，$f(t-t_0)=0$，相当于$f(t-t_0) = u(t-t_0) f(t-t_0)$
	$f (t\pm t_0)\mathop{\longleftrightarrow}\limits^{\mathcal L} e^{\pm s t_0}F (s)$
	$e^{\mp a t}f (t)\mathop{\longleftrightarrow}\limits^{\mathcal L}F (s\pm a)$
	- 注：
		逆变换时$u(t-t_0)$不可省略
		$e^{-st_0}\frac{s}{s^+1}$的${\mathcal L}^{-1}$为${\rm cos}(t-t_0) u(t-t_0)$
3. 微分性质：
	$f'(t)\mathop{\longleftrightarrow}\limits^{\mathcal L}sF(s)-f(0)$
	$f^{(n)}(t)\mathop{\longleftrightarrow}\limits^{\mathcal L}s^nF(s)-s^{n-1}f(0)-s^{n-2}f'(0)-...-s^0f^{(n-1)}(0)$
	$-tf(t)\mathop{\longleftrightarrow}\limits^{\mathcal L}F'(s)$
	$(-t)^nf(t)\mathop{\longleftrightarrow}\limits^{\mathcal L}F{(n)}(s)$
4. 积分性质：
	$\displaystyle{\mathcal L}[\int_{0}^{t}f (t){\rm d}t] = \frac{1}{s}F(s)$
	$\displaystyle\frac{f (t)}{t} \mathop{\longleftrightarrow}\limits^{\mathcal L} \int_{s}^{+\infty}F(s){\rm d}s$
	$\displaystyle\int_{0}^{+\infty}\frac{f (t)}{t}{\rm d}t = \int_{0}^{+\infty}F(s){\rm d}s$
5. 相似性：
	$\displaystyle f(at)\mathop{\longleftrightarrow}\limits^{\mathcal L}\frac{1}{a}F (\frac{s}{a})$
