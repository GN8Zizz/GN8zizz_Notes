## **随机试验**
通过研究随机试验来研究随机现象的。随机试验既包括科学实验，也包括对某一特征的观察
- 定义：
    若试验满足以下三个条件
	1. 试验可以在相同条件下重复进行
	2. 每次试验的可能结果不止一个，并且能事先明确试验的所有可能结果
	3. 进行一次试验之前不能确定哪一个结果会出现
	则称之为随机试验，简称试验，记作 $E$
- - - 
## **样本空间**
- 定义：
	- 样本空间：随机试验所有可能结果组成的集合，记作 $S$ 或 $\Omega$
	- 样本点：随机试验的每个结果，记作 $e$
	样本空间 $S$ 是由全体样本点 $e$ 构成的集合：$S = \{e\}$
- 求样本空间的方法：
	- 列举法：有限个样本点
	- 描述法：无限个样本点，用数学表达式描述
- - -
## **随机事件**
- 定义：
	- 随机事件：样本空间 $S$ 的子集称为 $E$ 的随机事件，简称事件，记作 $A,B$ 等
	- 事件发生：在每次试验中，当且仅当随机事件的一个样本点出现，反之称不发生
	- 基本事件：由一个样本点组成的单点集
	- 必然事件：样本空间 $S$ 包含所有样本点，在每次试验中它总发生
	- 不可能事件：空集∅不包含任何样本点，在每次试验中它都不发生
- - -
## **事件关系运算**
| <mark style="background: #FFB8EBA6;">关系</mark> | 集合论表示                                 | 概率论含义                                             |
| ---------------------------------------------- | ------------------------------------- | ------------------------------------------------- |
| 包含                                             | $A \subset B$                         | 事件 $A$ 发生必然导致事件 $B$ 发生                            |
| 相等                                             | $A = B$                               | 事件 $A$ 发生必然导致事件 $B$ 发生，反之亦然                       |
| 互斥</br>(互不相容)                                  | $AB = \varnothing$                    | 事件 $A$ 和 $B$ 不可能同时发生                              |
| 对立                                             | $AB = \varnothing$ </br>$A\cup B = S$ | 在一次试验中，事件 $A$ 和 $B$ 有且只有一个发生，记作 $B = \overline A$ |
- 推广：
	- 事件组 $A_1,A_2,...,A_n$ 中任意两个事件互不相容，称这些事件两两互斥或两两互不相容
		即 $A_iA_j = \varnothing\quad i\ne j\quad i,j=1,2,...$
	- 在一次试验中，基本事件是两两互不相容的
	

| <mark style="background: #FFB8EBA6;">运算</mark>  | 集合论表示                        | 概率论含义                                  |
| ---- | ---------------------------- | -------------------------------------- |
| 和(并) | $A \cup B$ 或</br>$A+B$       | 两个事件 $A,B$ 中，至少有一个事件发生                 |
| 积(交) | $A\cap B$ 或</br>$AB$         | 当且仅当事件 $A$ 与 $B$ 同时发生时，事件 $A\cap B$ 发生 |
| 差    | $A-B$ 或</br>$A \overline{B}$ | 当且仅当事件 $A$ 发生，$B$ 不发生时，$A-B$ 发生        |
- 推广：
	- $n$ 个事件 $A_1,A_2,...,A_n$ 的和、积事件：
		$\displaystyle A_1 \cup A_2 \cup ... \cup A_n = \bigcup_{i=1}^{n}A_i$
		$\displaystyle A_1 \cap A_2 \cap ... \cap A_n = \bigcap_{i=1}^{n}A_i$
	- 可列个事件 $A_1,A_2,...,A_n$ 的和、积事件：
		$\displaystyle A_1 \cup A_2 \cup ... \cup A_n \cup... = \bigcup_{i=1}^{\infty}A_i$
		$\displaystyle A_1 \cap A_2 \cap ... \cap A_n \cap... = \bigcap_{i=1}^{\infty}A_i$

- 事件的运算法则：
	与集合运算法则一致，但从"事件发生"的角度理解
	1. 吸收律：
		若 $A\subset B$，则 $A\cup B = B;AB=A;\overline{A}\supset\overline{B}$
	2. 交换律：
		$A\cup B = B\cup A$
		$AB=BA$
	3. 结合律：
		$(A\cup B)\cup C = A\cup(B\cup C)$
		$(AB)C=A(BC)$
	4. 分配率：
		$A(B\cup C) = AB\cup AC$
		$A\cup BC = (A\cup B)(A\cup C)$
		$A(B-C) = AB-AC$
	5. 对偶律(德摩根律)：
		$\overline{A\cup B} = \overline{A}\hspace{1mm}\overline{B}$
		$\overline{A\cup B\cup C} = \overline{A}\hspace{1mm}\overline{B}\hspace{1mm}\overline{C}$
		$\overline{AB} = \overline{A}\cup\overline{B}$
		$\overline{ABC} = \overline{A}\cup\overline{B}\cup\overline{C}$
- 事件运算顺序：先逆运算，再交运算，后并、差运算

## **频率与概率**
- 概率的描述性定义：
	称随机事件 $A$ 发生的可能性大小的度量 (非负值) 为事件 $A$ 发生的概率
- 概率的统计性定义(频率)：
	在相同条件下，进行了 $n$ 次试验，在这 $n$ 次试验中，事件 $A$ 发生的次数 $n_A$，称为事件 $A$ 发生的频数，比值 $\displaystyle\frac{n_A}{n}$ 称为事件 $A$ 发生的频率，记作 $f_n(A)$
	- 频率的基本性质：
		1. $0\le f_n(A) \le 1$
		2. $f_n(S) = 1$
		3. 若 $A_1,A_2,...,A_k$ 是两两互不相容的事件，则
			$f(A_1\cup A_2\cup ... \cup A_k) = f(A_1)+f(A_2)+...+f(A_k)$
		- 频率的稳定性：
			$n\rightarrow\infty,f_n(A)\rightarrow常数$
- 概率的公理化定义：
	设 $E$ 是随机试验，$S$ 是它的样本空间；对于 $E$ 的每一个事件 $A$ 赋予一个实数，记作 $P(A)$，若集合函数 $P(\cdot)$ 满足：
	1. 非负性：
		对每个事件 $A$，有 $P(A)\ge0$
	2. 规范性：
		对必然事件 $S$，有 $P(S) = 1$
	3. 可列可加性：
		设 $A_1,A_2,...,A_n,...$ 是两两互不相容的事件，即 $A_iA_j = \varnothing\quad i\ne j\quad i,j=1,2,...$
		有 $P(A_1\cup A_2\cup...\cup A_n\cup ...) = P(A_1)+P(A_2)+...+P(A_n)+...$

- 概率的重要性质：
	1. $P(\varnothing)=0$
	2. 有限可加性：
		设 $A_1,A_2,...,A_n$ 是两两互不相容的事件，即 $A_iA_j = \varnothing\quad i\ne j\quad i,j=1,2,...$
		有 $P(A_1\cup A_2\cup...\cup A_n) = P(A_1)+P(A_2)+...+P(A_n)$
	3. 减法公式：
		设 $A,B$ 是任意两个事件，则
		$P(A\overline{B})=P(A-B) = P(A)-P(AB)$
	4. 单调性：
		设 $A,B$ 是两个事件，若 $B\subset A$，则
		$P(A-B) = P(A)-P(B)\quad P(A)\ge P(B)$
	5. 有界性：
		对任一事件 $A$，有
		$P(A)\le1$
	6. 逆事件概率：
		对任一事件 $A$，有
		$P(\overline{A}) = 1-P(A)$
	7. 加法公式：
		对于任意两个事件 $A,B$，有
		$P(A\cup B) = P(A)+P(B)-P(AB)$
		- 推广：
			$P(A\cup B\cup C) = P(A)+P(B)+P(C)-P(AB)-P(AC)-P(BC)+P(ABC)$

## **古典概型**
- 古典概型定义：
	若随机试验 $E$ 满足：
	1. 样本空间 $S$ 只含有有限个样本点，$S = \{e_1,e_2,...,e_n\}$
	2. 每个基本事件(样本点) 发生的可能性相同
	则称此随机试验的概率模型为等可能概型，也称古典概型
- 古典概型中，事件 $A=\{e_{i1},e_{i2},...,e_{ik}\}$ 发生的概率为
	$P(A) = \frac{k}{n}=\frac{A包含的基本事件数}{S中基本事件的总数}$
- - -
## **几何概型**
- 几何概型定义：
	若随机试验 $E$ 满足：
	1. 样本空间 $S$ 是 $R^n(n=1,2,3)$ 中一个可度量的几何区域
	2. 每个样本点出现的概率相等，即样本点落入 $S$ 某一可度量的子区域 $A$ 的可能性大小与 $A$ 的几何度量成正比，而与 $A$ 的位置及形状无关
	则称此随机试验的概率模型为几何概型
- 几何概型中，事件 $A=\{样本点落入区域A\}$ 发生的概率为
	$P(A) = \frac{A的几何度量}{S的几何度量}$
- - -
## **条件概率**
- 条件概率的定义：
	设 $A,B$ 是两个事件，且 $P(A)>0$，称 $P(B|A)=\displaystyle\frac{P(AB)}{P(A)}$ 为在 $A$ 发生的条件下事件 $B$ 发生的条件概率
- 条件概率的性质：
	当 $P(A)>0$ 时
	1. $P(B|A)>0$
	2. $P(S|A) = 1;P(\varnothing|A)=0$
	3. $P(B\cup C|A) = P(B|A)+P(C|A)-P(BC|A)$
	4. 当 $B,C$ 互不相容时，$P(B\cup C|A)=P(B|A)+P(C|A)$
	5. $P(B-C|A) = P(B|A)-P(BC|A)$
	6. $P(\overline{B}|A) = 1-P(B|A)$
- - -
## **乘法公式**
- 设 $P(A)>0$，则有
	$P(AB)=P(A)P(B|A)$
	称为乘法公式
	- 推广：
		1. 设 $A,B,C$ 为事件，且 $P(AB)>0$，则有
			$P(ABC) = P(A)P(B|A)P(C|AB)$
		2. 设 $A_1,A_2,...,A_n$ 为 $n$ 个事件，$n>2$，且 $P(A_1A_2...A_{n-1})>0$，则有
			$P(A_1A_2...A_{n-1}A_n)=P(A_1)P(A_2|A_1)P(A_3|A_1A_2)...P(A_n|A_1A_2...A_{n-1})$
	- 注：
		- 注意事件发生先后次序，$A_i$ 先于 $A_{i+1}$ 发生
		- 主要用来计算没有相互独立性的若干事件的积事件的概率
- - -
## **全概率公式**
![[Drawing 2025-11-04 18.54.16.excalidraw|300]]
- 定理：
	设试验 $E$ 的样本空间 $S$，$A$ 为 $E$ 的事件，$B_1,B_2,...,B_n$ 为 $S$ 的一个划分，且 $P(B_i)>0\quad(i = 1,2,...,n)$，则
	$P(A) = P(B_1)\cdot P(A|B_1)+P(B_2)\cdot P(A|B_2)+...+P(B_n)\cdot P(A|B_n)$
	$\hspace{2.2mm}\qquad$<mark style="background: #FFB8EBA6;">$=\displaystyle\sum\limits_{i=1}^{n}P(B_i)\cdot P(A|B_i)$</mark>
	- 注：
		- 若将 $A$ 看为"果"，$B_1,B_2,...,B_n$ 看为"因"
			全概率公式反应的是"由因求果"的概率问题
		- 使用时需对涉及到的事件赋予概率符号
		- 关键是找到导致事件 $B$ 发生的样本空间的一个划分
### 样本空间的划分
- 定义：
	设 $S$ 为试验 $E$ 的样本空间，$B_1,B_2,...,B_n$ 为 $E$ 的一组事件，若
	1. $B_iB_j = \varnothing\quad(i\ne j\quad i,j=1,2,...,n)$
		即两两互不相容
	2. $B_1\cup B_2\cup ...\cup B_n = S$
	则称 $B_1,B_2,...,B_n$ 为样本空间 $S$ 的一个划分(或完备事件组)
	- 注：
		- 若 $B_1,B_2,...,B_n$ 为样本空间 $S$ 的一个划分，则对每次试验，事件 $B_1,B_2,...,B_n$ 中必有一个且仅有一个发生
		- 样本空间的划分一般不唯一
## **贝叶斯公式**
- 定理：
	设试验 $E$ 的样本空间为 $S$，$A$ 为 $E$ 的事件，$B_1,B_2,...,B_n$ 为 $S$ 的一个划分，且 $P(A)>0,P(B_i)>0\quad(i=1,2,...,n)$，则
	$\displaystyle P(B_i|A) = \frac{P(AB_i)}{P(A)} = \frac{P(B_i)P(A|B_i)}{\displaystyle\sum_{j=1}^{n}P(B_j)P(A|B_j)}\quad(i=1,2,...,n)$
	- 注：
		- 若将 $A$ 看为"果"，$B_1,B_2,...,B_n$ 看为"因"
			贝叶斯公式反应的是"由果求因"的概率问题

## **独立性**
1. 描述性定义：
	设 $A,B$ 为两个事件，若其中任何一个事件发生的概率不受另一个事件发生与否的影响，则称 $A$ 与 $B$ 相互独立
	$P(B|A) = P(B)\quad P(B|\overline{A}) = P(B)$
	$P(A|B) = P(A)\quad P(A|\overline{B}) = P(A)$
2. 数学定义：
	设 $A,B$ 为两个事件，若满足
	$P(AB) = P(A)\cdot P(B)$
	则称事件 $A,B$ 相互独立，简称独立
	- 注：
		1. 两事件相互独立不能用图示说明
		2. 两事件互相独立与互不相容不能同时成立
		3. 必然事件与不可能事件与任意事件相互独立
### 定理1
设 $A,B$ 是两事件，且 $P(A)>0$，若 $A,B$ 相互独立，则 $P(B|A) = P(B)$，反之亦然
### 定理2
若事件 $A$ 与 $B$ 相互独立，则下列事件也相互独立：
$A$ 与 $\overline{B}$ 、$\overline{A}$ 与 $B$、$\overline{A}$ 与 $\overline{B}$
## **三事件独立性**
1. 三事件两两独立
	若 $\left\{\begin{array}{l} P(AB) = P(A)P(B) \\ P(AC)=P(A)P(C) \\ P(BC)=P(B)P(C) \end{array}\right.$，则称事件 $A,B,C$ 两两独立
2. 三事件互相独立
	若 $\left\{\begin{array}{l} P(AB) = P(A)P(B) \\ P(AC)=P(A)P(C) \\ P(BC)=P(B)P(C) \\ P(ABC) = P(A)P(B)P(C) \end{array}\right.$，则称事件 $A,B,C$ 相互独立
- 相互独立 $\Rightarrow$ 两两独立
## **$n$ 个事件的独立性**
- 定义：
	设 $A_1,A_2,...,A_n$ 是 $n(n>2)$ 个事件，若对于其中任意 $2$ 个、$3$ 个、$...$ 、任意 $n$ 个事件的概率，都等于各事件的乘积，则称事件 $A_1,A_2,...,A_n$ 相互独立
- 推论：
	$A_1,A_2,...,A_n$ 相互独立
	$\Leftrightarrow$ 任意 $k(2\le k \le n)$ 个事件，$P(A_1A_2...A_k)=P(A_1)P(A_2)...P(A_k)$
	$\Leftrightarrow$ 它们中任意一部分事件换成各自的对立事件所得到的 $n$ 个事件相互独立
## **独立性的判定**
1. 直观性判定：
	若试验独立，则其结果必然相互独立，即根据事件的实际意义去判断
2. 利用定义与定理
